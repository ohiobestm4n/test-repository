Catatan HTML & CSS

(dijelaskan di kelas)

  • HTML

    HTML berfungsi sebagai kerangka dari sebuah website. Contohnya Instagram memiliki direct message, dan juga post-post. HTML mengatur fitur-fitur dari sebuah website. CSS digunakan untuk memberikan font-size, font-color, warna, dan gaya. Sebuah situs dapat dibuat tanpa CSS, namun CSS penting untuk membuat situs menjadi lebih bagus dan menarik.

  • Sintaks

    Sintaks adalah sebuah aturan penulisan dalam dunia pemrograman. Sintaks dari bahasa-bahasa pemrograman berbeda-beda. Aturan pertama penulisan kode HTML adalah mengapit teks dengan tag. h1-h6 berfungsi sebagai judul, h1 paling besar (tag untuk judul). Ketika menulis tag dalam HTML, tag pembuka ditulis langsung. Namun saat menulis tag penutup, harus disertakan garis miring yang berada di tombol sama dengan tanda tanya.
